# **App Name**: Liquidfolio

## Core Features:

- Hero Section: Display a brief introduction, title, and a captivating visual element.
- Skills Showcase: Visually present key skills using progress bars and icons.
- Projects Gallery: Showcase past projects with descriptions, images, and links.
- Education Timeline: Illustrate educational background with institutions and degrees.
- Certifications Display: List certifications with issuing organizations and dates.
- Basic Contact: Display Gmail and icons for GitHub and LinkedIn.
- Admin Area for Contact Management: Protected admin area to manage contacts from Firestore.

## Style Guidelines:

- Primary color: Light cyan (#E0FFFF) for a bright, airy feel.
- Background color: Very light gray (#F5F5F5) to provide a clean backdrop.
- Accent color: Sky blue (#87CEEB) for interactive elements and highlights.
- Font pairing: 'Nunito' (sans-serif) for headlines and 'Lato' (sans-serif) for body text, providing a modern and readable combination.
- Consistent, minimalistic line-style icons that match the font weight.
- Single-page layout with smooth anchor navigation and a responsive grid system.
- Subtle transitions and animations, respecting `prefers-reduced-motion`.
